# To-Do Application

## Structure
- frontend → Next.js app  
- backend → Spring Boot app  
- db → PostgreSQL scripts (tables, seed data)

## How to Start
1. Clone the repo  
2. cd TO-DO-App  
3. cd frontend → Run: npm run dev  
4. cd backend → Run: mvn spring-boot:run  
5. Database: PostgreSQL
